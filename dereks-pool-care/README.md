# Derek's Complete Pool Care Website

A professional, high-conversion marketing website for Derek's Complete Pool Care - a 5-star rated pool maintenance and repair service serving Long Island, NY.

## Features

- **Modern Tech Stack**: Next.js 15+ with TypeScript and Tailwind CSS
- **Beautiful UI**: shadcn/ui components with custom pool theme
- **SEO Optimized**: Comprehensive metadata and structured data for search engines
- **Contact Form**: Full validation with email notifications via Resend
- **Rate Limiting**: Built-in protection against spam submissions
- **Responsive Design**: Mobile-first design that works on all devices
- **Google Maps Integration**: Interactive map showing business location
- **Social Proof**: Customer testimonials and 5-star ratings
- **Portfolio Gallery**: Showcase of completed projects with before/after views

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd dereks-pool-care
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env.local
```

4. Edit `.env.local` and add your API keys:
   - **RESEND_API_KEY**: Sign up at [resend.com](https://resend.com)
   - **NEXT_PUBLIC_GOOGLE_MAPS_API_KEY**: Get from [Google Cloud Console](https://console.cloud.google.com/google/maps-apis)
   - **UPSTASH_REDIS_REST_URL** & **UPSTASH_REDIS_REST_TOKEN** (optional): Sign up at [upstash.com](https://upstash.com)

### Development

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Building for Production

```bash
npm run build
npm run start
```

## Configuration

### Email Setup

The contact form uses Resend for email delivery. Configure these environment variables:
- `RESEND_API_KEY`: Your Resend API key
- `BUSINESS_EMAIL`: Email address where contact form submissions are sent

### Google Maps

To display the interactive map:
1. Get a Google Maps API key from [Google Cloud Console](https://console.cloud.google.com)
2. Enable the Maps Embed API
3. Add the key to `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY`

If no API key is provided, a fallback UI with a "Get Directions" link will be shown.

### Rate Limiting

Rate limiting protects the contact form from spam:
- **With Upstash Redis**: Persistent rate limiting across serverless deployments
- **Without Upstash**: In-memory rate limiting (resets on deployment)

Default limit: 3 submissions per hour per IP address

## Customization

### Business Information

Edit `src/data/business.ts` to update:
- Company name, tagline, and description
- Contact information (phone, email, address)
- Business hours
- Social media links
- Service areas

### Services

Edit `src/data/services.ts` to modify service offerings, pricing, and features.

### Testimonials

Edit `src/data/testimonials.ts` to add or update customer reviews.

### Portfolio

Edit `src/data/portfolio.ts` to showcase completed projects.

### Styling

The site uses a custom "pool" color theme defined in `tailwind.config.ts`:
- `pool-50`: Light blue background (#f0f9ff)
- `pool-500`: Primary sky blue (#0ea5e9)
- `pool-900`: Deep blue dark (#0c4a6e)
